# Shortlink

A project that converts URLs into shortened URLs like bit.ly or t.co by Twitter or Facebook.



# Installation

1. git clone https://github.com/pluhmylkin/shortlink.git
2. cd shortlink/
3. docker-compose up
4. Go to http://localhost:3008
